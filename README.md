## 🧠 What this app does

fireworks-skill-memory keeps skill history for Claude Code in one place. It stores past skill use so Claude Code can keep track of what worked before. This helps you keep a steady workflow and avoids repeating the same setup steps.

Use it when you want:

- A record of skill use over time
- Less repeated setup work
- A simple way to keep context across sessions
- More consistent results from Claude Code

## 🧩 How it works

The app keeps per-skill memory. That means each skill can keep its own record instead of mixing with other skills.

Typical use:

- You run a skill in Claude Code
- The app saves the result or context
- The next time you use that skill, the app can recall past details
- You get a more stable setup each time

This is useful when you use the same skill many times and want the same starting point.

## 🛠️ Basic use

To use fireworks-skill-memory day to day:

1. Open Claude Code.
2. Open the memory app.
3. Pick the skill you want to use.
4. Run your task in Claude Code.
5. Save the result when you finish.
6. Reuse the same skill later and load the stored memory.

If you work with more than one skill, keep one memory entry per skill. This keeps your history clean and easy to follow.
